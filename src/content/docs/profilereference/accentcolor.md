---
title: accentColor
---

## Availability
* Setup Manager since version 1.0

## Description

Sets the accent color for buttons, progress bar, SF Symbol icons, and other UI elements. Use this to match branding. Color is encoded as a six digit hex code, e.g. #FF0088.

## Key type

string

## Default value

_undefined_

## Valid values

Any valid [color notation](/docmirror/colors).

## Examples
```xml
<key>accentColor</key>
<string>#FF00AA</string>
```
```xml
<key>accentColor</key>
<string>##green</string>
```
